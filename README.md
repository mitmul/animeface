# animeface

This repository is a part of Chainer tutorial handouts. It shows

- How to write your original dataset class in Chainer
- How to fine-tune a pre-trained model on another dataset

## Requirement

- Chainer 1.23.0+
- PIL 4.0.0+
- NumPy 1.12.0+

## Prepare dataset

Just run

```bash
bash prepare.sh
```
